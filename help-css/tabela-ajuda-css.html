Tabela Ajuda CSS

Propriedade Função
Propriedades para texto
text-align: Define o alinhamento do texto.
font-size: Define o tamanho da fonte.
font-weight: Define a tonalidade da fonte.
font-style: Define o estilo da fonte(itálico por exemplo).
Propriedades gerais
color: Define a cor to texto de um elemento.
background-color: Define a corde fundo de um elemento da página.
Bordas
border: Define as bordas.
Seletores de Borda
border-top: Define apenas a borda de cima.
border-left: Define apenas a borda esquerda.
border-bottom: Define apenas a borda de baixo.
border-right: Define apenas a borda direita.
border-color: Define a cor das bordas em geral.
Seletores de cor da Borda
border-top-color: Define apenas a cor da borda de cima.
border-left-color: Define apenas a cor da borda esquerda.
border-bottom-color: Define apenas a cor da borda de baixo.
border-right-color: Define apenas a cor da borda direita.
border-radius: Arredonda as bordas.
Seletores de arredondamento da Borda
border-top-radius: Arredonda apenas as bordas de cima.
border-left-radius: Arredonda apenas as bordas esquerdas.
border-bottom-radius: Arredonda apenas as bordas de baixo.
border-right-radius: Arredonda apenas as bordas direitas.
border-style: Define o estilo das bordas,Para exemplos clique aqui
Seletores de arredondamento da Borda
border-top-style: Altera apenas o estilo da borda de cima.
border-left-style: Altera apenas o estilo da borda esquerda.
border-bottom-style: Altera apenas o estilo da borda de baixo.
border-right-style: Altera apenas o estilo da borda direita.
Para Listas
list-style-type: Define o estilo da lista,exemplo figuras,algarismos romanos,etc.
list-style-image: Define uma imagem para enumerar a lista
list-style-position: Define a posição do estilo da lista
