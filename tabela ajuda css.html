Tabela Ajuda CSS



Propriedades Função
Propriedades para texto
text-align: Define o alinhamento do texto.
font-size: Define o tamanho da fonte.
font-weight: Define a tonalidade da fonte.
font-style: Define o estilo da fonte(itálico por exemplo).
Propriedades gerais
color: Define a cor dos elementos como textos,links,bordas,etc...
background-color: Define a corde fundo de algo,pode ser do corpo da página,
de uma tabela ou de lugares específicos da página.
Bordas
border: Define uma borda.
border-top: Define a borda de cima.
border-left: Define a borda esquerda.
border-bottom: Define a borda de baixo.
border-right: Define a borda direita.
border-color: Define a cor da borda em geral.
border-top-color: Define a cor da borda de cima.
border-left-color: Define a cor da borda esquerda.
border-bottom-color: Define a cor da borda de baixo.
border-right-color: Define a cor da borda direita.
border-radius: Arredonda as bordas.
border-top-radius: Arredonda a borda de cima.
border-left-radius: Arredonda a borda esquerda.
border-bottom-radius: Arredonda a borda de baixo.
border-right-radius: Arredonda a borda direita.
Para Listas
list-style-type: Define o tipo do estilo da lista,exemplo figuras,algarismos romanos,etc.
list-style-image: Define uma imagem para enumerar a lista
list-style-position: Define a posição do estino da lista
