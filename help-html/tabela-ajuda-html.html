Tabela Ajuda HTML


Códigos HTML
Tag Fechamento Função
Basico
<html> </html> Define a raiz do documento html
<head> </head> Tag onde são postas as informações da página.
Dentre essas informações estão:
<title> </title> Define o título da página (que é visto na guia)
<link> Não possui Faz a ligação de algum arquivo na página (um arquivo css, um icone, entre outros).
<meta> Não possui Define metadados da página como autor,descrição,codificação de caracteres,etc
Observação: as definições devem ser feitas por meio de atributos
<body> </body> É o "corpo" da página, aqui onde são colocados todos os elementos visíveis.
<header> </header> Define o cabeçalho para a página
<main> </main> Define o conteúdo principal da página
<section> </section> Cria uma seção na página
<div> </div> Cria uma divisão na página
<a> </a> Adiciona um hiperlink, para exemplos clique aqui
<footer> </footer> Define o rodapé da página
<!-- --> Adiciona um comentário (só pode ser visualizado no código fonte)
Formatação de Texto
<i> </i> Deixa o texto em itálico (forma)
<em> </em> Deixa o texto enfatizado (itálico, porém semântico)
<b> </b> Deixa o texto em negrito (forma)
<strong> </strong> Deixa o texto em destaque (negrito, porém semântico)
<mark> </mark> Deixa o texto em destaque (marcado,semelhante a canetinhas marca texto).
<small> </small> Deixa o texto pequeno
<del> </del> Indica um texto deletado.
<ins> </ins> Indica um texto inserido.
<u> </u> Sublinha o texto.
<sup> </sup> Indica um texto sobrescrito
<sub> </sub> Indica um texto subscrito
<code> </code> Indica um trecho de código inline.
<pre> </pre> Indica um texto pré formatado (mantém todos os espaços e quebras de linha).
<abrr> </abrr> Indica uma abreviatura (com definição).
A definição deve ser definida pelo atributo title="definição"
<marquee> </marquee> Faz o texto (ou imagem) rodar na página
Para ver os exemplos relacionados a texto clique aqui
Espaçamento
<p> </p> Define um parágrafo
<br> Não possui Indica uma quebra de linha (enter).
<hr> Não possui Cria uma linha horizontal que divide todo o elemento em que está presente
Para exemplos de espaçamento clique aqui
Listas
<li> </li> Indica um item em uma lista.
<ul> </ul> Indica uma lista não enumerada.
Atenção!!! A tag <li> deve ser usada dentro da tag <ul> para que ela funcione
<ol> </ol> Indica uma lista enumerada
Atenção!!! A tag <li> deve ser usada dentro da tag <ol> para que ela funcione
<dt> </dt> Define um item (como um título)
<dd> </dd> Define um item secundário (como um subtítulo)
Para vez exemplos das tags de listagem clique aqui
Mídia
<img> </img> Adiciona uma imagem,Para exemplos clique aqui
<audio> </audio> Adiciona um audio na página
<video> </video> Adiciona um video na página
Atenção!!! O caminho para a midia deve ser especificado no atributo src="nome do arquivo da imagem"
Tabelas
<table> </table> Cria uma tabela.
<thead> </thead> Indica um cabeçalho para a tabela.
<tbody> </tbody> Indica o corpo da tabela.
<tr> </tr> Indica uma linha na tabela.
<th> </th> Indica uma coluna de cabeçalho para a tabela.
<td> </td> Indica uma coluna para a tabela.
Formulários
<form> </form> Cria um formulário,em outras palavras,ele agrupa todas as tags seguintes que recebem dados
<input> Não possui Recebe algum determinado tipo de dado, para exemplos clique aqui
Atenção!!! O tipo de dado a ser coletado deve ser especificado pelo atributo type="tipo de dado"
<select> </select> Cria um quadro de seleção
Para adicionar as opções use a tag abaixo
<option> </option> Indica uma opção para algum campo input.
<fieldset> </fieldset> Cria um elemento para agrupar alguns inputs, pode ser usado para organizar o formulário.
<legend> </legend> Indica um título para um <fieldset>
<textarea> </textarea> Cria uma área que recolhe texto.
Para exemplos clique aqui
Títulos
Tag Fechamento Função
<h1> </h1> Indica um título de nível 1
<h2> </h2> Indica um título de nível 2
<h3> </h3> Indica um título de nível 3
<h4> </h4> Indica um título de nível 4
<h5> </h5> Indica um título de nível 5
<h6> </h6> Indica um título de nível 6
Os títulos seguem uma ordem hierárquica onde o nível 1 é o mais importante e o nível 6 o menos importante.
